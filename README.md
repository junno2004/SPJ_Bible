# SPJ 성경 (Korean Bible App)

Next.js로 제작된 현대적인 디자인의 한글 성경 웹 애플리케이션입니다.
깔끔한 인터페이스로 성경을 읽고, 검색하고, 묵상할 수 있는 환경을 제공합니다.

## ✨ 주요 기능 (Features)

1.  **📖 성경 읽기 (Bible Reading)**
    *   구약과 신약의 모든 권, 장, 절을 쉽게 찾아 읽을 수 있습니다.
    *   모든 성경 책 이름이 한글로 제공됩니다 (예: 창세기, 요한복음).

2.  **🔍 성경 검색 (Search)**
    *   원하는 단어나 구절을 빠르게 검색할 수 있습니다.
    *   검색 결과에서 키워드가 하이라이트되어 쉽게 찾을 수 있습니다.

3.  **📅 오늘의 말씀 (Verse of the Day)**
    *   매일 새로운 성경 구절을 메인 화면에서 추천해 줍니다.

4.  **🌙 다크 모드 (Dark Mode)**
    *   눈이 편안한 다크 모드를 지원합니다.
    *   우측 상단의 해/달 아이콘으로 테마를 전환할 수 있습니다.

5.  **✍️ 말씀 형광펜 (Highlighting)**
    *   마음에 드는 구절을 클릭하면 형광펜처럼 색칠(하이라이트)할 수 있습니다.
    *   표시한 내용은 브라우저에 자동 저장되어 나중에 다시 와도 유지됩니다.

## 🛠️ 기술 스택 (Tech Stack)

*   **Framework**: Next.js 14+ (App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS (v4)
*   **Theme**: next-themes
*   **Icons**: Lucide React

## 🚀 실행 방법 (How to Run)

프로젝트를 로컬에서 실행하려면 다음 명령어를 입력하세요.

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:3000` (또는 터미널에 표시된 포트)으로 접속하세요.
